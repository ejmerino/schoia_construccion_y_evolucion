# **App Name**: MallaMentor

## Core Features:

- Interactive Curriculum Grid: Interactive curriculum grid that provides a clear visual representation of the student's academic progress. Lets students drag and drop courses to plan their semesters.
- Course Completion Validation: A student can select completed courses, automatically validating the completion of prerequisites, ensuring academic compliance.
- Available Courses Display: Personalized recommendations using studentâ€™s progress. Students see which courses are available each semester.
- Semester Cost Calculator: Estimate cost per semester based on selected courses, dynamically adjusted according to the student's quintile. Implements quintile-based fee logic, as passed in by the user.
- Schedule Prediction: A tool predicting the future course schedules. This system will analyze historical scheduling data, course demands, and enrollment trends of professors in previous semesters to offer reliable course schedule estimations, with a confidence measure, which will aid in academic planning.
- Difficulty Statistics: Analysis of grades data and difficulty ratings from students (using prior grades) using grades of current students, generating statistics only visible in admin dashboard.
- Curriculum Export: Option to export planned curriculum as PDF.

## Style Guidelines:

- Primary color: ESPE red (#E62B3E) used for headers and buttons to maintain university branding and provide strong visual impact.
- Success color: ESPE green (#036A3F) to indicate approved subjects and completed requirements, symbolizing success.
- Warning color: ESPE yellow (#FFE811) for available subjects, signaling readiness and opportunity.
- Background color: Off-white (#FEFFFF) used as the base to ensure content readability and a clean interface. Color scheme will have a dark mode variation using a dark background.
- Body and headline font: 'Inter' (sans-serif) ensures optimal readability across the platform.
- Use a set of modern and intuitive icons to represent various actions and modules within the platform. All icons are designed with simplicity and clarity.
- Dashboard uses a modern card-based layout, creating an engaging and easy-to-navigate platform with key academic information at a glance. Responsive across devices.
- Implement smooth transitions and subtle animations to improve user experience, e.g., when loading data or updating the curriculum plan.
